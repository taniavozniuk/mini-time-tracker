# Mini Time Tracker — Frontend

## Overview

Frontend application for the **Mini Time Tracker**.
Built with **Next.js**, **React**, and **TypeScript**.

Provides UI for creating time entries and viewing history.

---

## Tech Stack

- Next.js
- React
- TypeScript
- Material UI (MUI)
- Axios
- Day.js

---

## Architecture

Frontend follows a component-based architecture:

- **Components** — reusable UI elements
- **Pages** — route-based views
- **API layer** — communication with backend
- **Constants** — shared static values
- **Types** — TypeScript models

## Features

### Time Entry Form

- Date picker
- Project selection
- Hours input
- Description field
- Validation & error handling

### Entry History

- Grouped by date
- Daily totals
- Grand total

## Installation & Running the Project

### Clone the repository

```bash
git clone https://github.com/taniavozniuk/mini-time-tracker.git
cd mini-time-tracker
npm install
```

### Start development server

```bash
cd frontend
npm run dev
```

### API will be available at:

```bash
http://localhost:3000
```
